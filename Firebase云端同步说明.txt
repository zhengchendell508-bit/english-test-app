英语测试 APP — Firebase Lesson 云端同步最终版

本阶段已经完成
================
1. 家长后台打开时优先读取 Firestore 云端题库。
2. 后台点击“保存题库”时，同时保存：
   - Firestore 云端
   - 当前设备 localStorage 离线缓存
3. TXT / JS 导入完成后自动同步到 Firestore。
4. 批量设置“显示中文 / 播放声音”后自动同步到 Firestore。
5. 清空本部分或清空全部题库后自动同步到 Firestore。
6. 电脑和 iPad 打开孩子测试页面时，都优先读取同一份 Firestore 题库。
7. Lesson 下拉框根据云端实际 Lesson 数量生成。
8. 中文提示、英文答案和出题模式会跨设备保持一致。
9. 中文模式下，只要有英文答案，就始终显示小喇叭。
10. Firestore 或网络临时失败时，自动使用当前设备最近一次本地缓存。

首次使用步骤
============
1. 将压缩包中的全部文件覆盖上传到 GitHub 仓库根目录。
2. 等待 GitHub Pages 部署完成。
3. 在保存着旧题库的电脑上打开“家长 → 题库管理”。
4. 确认题库内容正确后，点击“保存题库”。
5. 顶部显示“✓ 已同步到云端”后，Firestore 中会自动创建：
   englishTestApp / lessonBank
6. 在 iPad 上重新打开测试页面，即可看到同样的 Lesson 和喇叭设置。

重要说明
========
- 目前只同步 Lesson 题库。
- 孩子名单、答案、页码、计时器和提交状态仍保存在各自设备。
- Firestore 当前是测试模式。测试完成后需要设置正式安全规则。
- 不需要在 Firebase 控制台手动点击“启动集合”；代码会自动创建集合和文档。
