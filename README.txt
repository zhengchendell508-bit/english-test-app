新概念英语答题 App（网页版）

文件：
- index.html
- style.css
- script.js
- data.js
- manifest.webmanifest
- sw.js

使用：
1. 将整个文件夹上传到 Replit、Netlify、GitHub Pages 或 Firebase Hosting。
2. 用 iPad Safari 打开网址。
3. 点击 Safari 的“分享”按钮。
4. 选择“添加到主屏幕”，之后就会像 App 一样打开。

功能：
- 单词150题、短语200题、句子200题
- 自动保存到本机浏览器
- 提交后自动批改
- 统计正确、错误、未答和分数
- 可以设置是否显示正确答案
- 支持跳转题号
