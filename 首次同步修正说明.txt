本修正版解决的问题
==================
旧版在 Firestore 已经存在“空白 Lesson 1”时，会把云端空白题库当作有效数据，
从而覆盖电脑浏览器里原来已有的题库，导致后台没有数据，孩子测试页也没有喇叭。

本版首次打开家长后台时会自动判断：
1. 电脑本地题库有实际内容；
2. 云端题库不存在，或者只有空白模板。

符合以上条件时，将电脑本地原题库自动上传到 Firestore，不再用空白云端数据覆盖电脑数据。

使用步骤
========
1. 用本压缩包完整覆盖 GitHub 仓库。
2. 等待 GitHub Pages 部署完成。
3. 必须先在“保存着原始题库数据的那台电脑”打开家长后台。
4. 顶部应显示：
   ✓ 已把电脑原题库首次同步到云端
5. 然后点击一次“保存题库”再次确认。
6. 最后在 iPad 重新打开孩子测试页。

注意
====
不要先在没有题库数据的 iPad 上进入后台执行保存。
首次迁移必须由仍然保存着原题库的电脑完成。
