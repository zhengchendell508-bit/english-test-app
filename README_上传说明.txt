英语测试 APP——TXT / JS 自动生成 Lesson 版

本次修改重点
============

1. 家长后台支持上传 TXT 和 JS。
2. 上传后自动分类：
   - 1 个英文词：单词
   - 2–5 个英文词：短语
   - 6 个及以上英文词：句子
3. 每类每 30 条自动生成一个 Lesson：
   - Lesson 1：第 1–30 条
   - Lesson 2：第 31–60 条
   - Lesson 3：第 61–90 条
4. 后续上传会接着现有资料继续排列。
5. 完全相同的英文内容会自动跳过，避免重复。
6. 最后一个 Lesson 不足 30 条时，会保留空白位置，后台显示实际数量。

TXT 格式
========

每行一题：

apple | 苹果
good morning | 早上好
How are you today? | 你今天怎么样？

也可以只写英文，但中文提示会暂时为空。

JS 格式
=======

支持这种文件：

const CHINESE_TRANSLATIONS = {
  "private": "私人的；私下的",
  "a private conversation": "一次私人谈话",
  "I went to the theatre last week.": "上星期我去看戏了。"
};

使用方法
========

1. 进入“家长”。
2. 打开“题库管理”。
3. 点击“上传 TXT / JS”。
4. 选择文件。
5. 系统自动分类、去重并生成 Lesson。
6. 检查内容后点击“保存题库”。

上传 GitHub
==========

将压缩包内所有文件上传到仓库根目录。
同名文件直接覆盖。
上传后刷新页面两次；如果仍看到旧版，可删除主屏幕旧图标后重新添加。
